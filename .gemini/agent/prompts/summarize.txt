# Context Summarization Prompt Template

Summarize the following conversation context into a compressed summary. The summary should capture the essential information while reducing token count.

## Summary Format

Your response should be structured as follows:

1. **Overview** (2-3 sentences): Brief description of what was discussed/accomplished
2. **Key Decisions**: Bullet list of important decisions made
3. **Architecture Changes**: Bullet list of structural or design changes
4. **Code Changes**: Brief list of files modified and what changed
5. **Next Steps**: Suggested follow-up actions

## Guidelines

- Be concise but comprehensive
- Preserve technical details that affect implementation
- Note any constraints or requirements established
- Capture the "why" behind decisions, not just the "what"
- Highlight any unresolved questions or issues
